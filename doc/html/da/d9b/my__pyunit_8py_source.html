181 class TestSuite(object):
182  '''A test suite is a composite test consisting of a number of TestCases.
183 
184  For use, create an instance of TestSuite, then add test case instances.
185  When all tests have been added, the suite can be passed to a test
186  runner, such as run(). It will run the individual test cases
187  in the order in which they were added, aggregating the results. When
188  subclassing, do not forget to call the base class constructor.
189  '''

191  self.suite = []
